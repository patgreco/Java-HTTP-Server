Arduino Robot Project

Creator: Patrick Greco

Overview

Completed as a CE (comprehensive examination) project for Dawson college, I was given a robot car as shown in the photo below. My task was attach an Arduino to the car as well as various sensors, as the robot was expected to complete a course containing three sections (see below).

UltraSonic Sensor

The robot was tasked to maintain a certain distance (50cm) to a wall using an ultrasonic sensor (which measures distance). It must navigate along the wall until it reached the next obstacle.

Remote Control

Next, the robot must switch to remote control mode, where the user will navigate the robot through the maze. The robot must not touch the walls of the maze.

Color Sensor

Lastly, the robot will switch to color sensor mode in order to navigate through a colored track. It must maintain itself on the blue line.
